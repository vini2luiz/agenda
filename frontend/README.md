# Frontend - Agenda de Contatos

Frontend desenvolvido com React, TypeScript e Vite.

## Instalação

```bash
npm install
```

## Execução

### Modo desenvolvimento:
```bash
npm run dev
```

O aplicativo estará disponível em `http://localhost:3000`

### Build para produção:
```bash
npm run build
```

### Preview da build:
```bash
npm run preview
```

## Funcionalidades

- ✅ Listagem de contatos
- ✅ Criação de contatos
- ✅ Edição de contatos
- ✅ Exclusão de contatos
- ✅ Busca por nome, e-mail ou telefone
